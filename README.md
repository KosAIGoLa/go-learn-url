## 学习路径建议

1. **第 1-2 周**: 完成 [A Tour of Go](https://go.dev/tour/) + [Go by Example](https://gobyexample.com/)
2. **第 3-4 周**: 阅读 [Effective Go](https://go.dev/doc/effective_go) + 完成 LeetCode 简单题
3. **第 5-6 周**: 学习 [Go 语言高级编程](https://books.studygolang.com/advanced-go-programming/)
4. **第 7-8 周**: 深入 [Go 语言设计与实现](https://draveness.me/golang/)
5. **持续**: 参与 [Go 夜读](https://github.com/talkgo/night) + 阅读源码
